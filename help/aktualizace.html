Aktualizace programu Ticketník

Program si sám heldá aktualizace při spuštění. Další možností je ruční vyhledání aktualizace kliknutím na tlačítko 'O Programu' z hlavního menu.

Aktualizace se primárně vyhledává na GitHub. V případě neúspěšné aktualizace z GitHub se program snaží aktualizovat ze WpsSlave v lokální Tieto síti (nutná VPN nebo být přímo v síti).
