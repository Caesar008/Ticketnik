O programu Ticketník {verze}

Program pro ukládání ticketů pro jejich snadný přehled a správu.

Výchozí jazyk rozhraní je angličtina, ale program podporuje více jazyků.

Program využívá knihovny fNbt, kterou vytvořil Matvei "fragmer" Stefarov.
Knihovna pracuje se soubory formátu Named Binary Tag, který vymyslel Markus "Notch" Persson.
